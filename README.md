# fonts
This repository is a collection of Greek fonts and font resources designed for use in typography, design, publishing, and digital projects that require accurate and high-quality Greek lettering.
It aims to make working with Greek text easier for designers, developers, researchers, and anyone interested in Greek typography.

What This Project Includes
  Greek fonts (modern, classical, or stylized)
  Unicode-compliant Greek characters
Support for:
  Monotonic Greek
  Polytonic Greek 
Font files in common formats (e.g. .ttf, .otf)
Documentation and usage examples (where available)
